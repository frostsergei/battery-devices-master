# battery-devices-master

## Setup Instruction 

1. Install nswag — automatic code generation tool.
  `npm install nswag -g`
2. Restore NuGet packages.
  `dotnet restore`
3. Build project.
  `make`
4. Run application.
  `./run` (Unix) or `dotnet run --project server -lp https`
